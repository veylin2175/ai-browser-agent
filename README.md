# AI Browser Agent

Автономный AI-агент для управления браузером на базе Playwright + LLM.  
Выполняет сложные многошаговые задачи в веб-браузере без предзаготовленных скриптов и хардкодных селекторов.

https://github.com/veylin2175/ai-browser-agent

## Возможности

- Полностью автономное выполнение задач по текстовому описанию
- Управление видимым Chromium-браузером (не headless)
- Поддержка persistent sessions (можно залогиниться вручную → агент продолжит)
- Действия: навигация, ввод текста, клик по элементам, нажатие клавиш
- Безопасность: запрашивает подтверждение перед потенциально деструктивными действиями
- Адаптация к ошибкам: после таймаута/неудачи пробует альтернативные шаги
- Минимальный контекст: отправляет в LLM только список интерактивных элементов (snapshot), а не всю страницу

## Примеры задач, которые агент успешно решает

- "Перейди на яндекс маркет, найди мне Айфон 17 и поставь цену от 95 000 до 100 000 рублей"
- "Зайди на ya.ru, найди AI browser agents 2026 и нажми поиск"
- "Открой Gmail и удали все письма от спама" (с подтверждением удаления)

## Архитектура

- **Браузер**: Playwright (persistent context, visible Chromium)
- **LLM**: Mistral (или любой OpenAI-совместимый провайдер: Groq, Together, DeepSeek и т.д.)
- **Извлечение элементов**: JavaScript-скрипт в Interpreter → список интерактивных элементов (индекс, селектор, role, name, disabled)
- **Агент**: цикл Step → LLM генерирует одно JSON-действие → Executor выполняет
- **Контекст**: короткая история последних действий + observation (URL, title, snippet текста, изменения)
- **Security layer**: перед кликом/вводом/нажатием клавиши проверяет текст элемента на ключевые слова ("оплатить", "удалить", "buy", "delete" и т.п.) → запрашивает подтверждение в терминале
- **Вывод**: чистый и понятный для пользователя (без логов, только действия вида "→ Ввожу текст ...")

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/veylin2175/ai-browser-agent.git
cd ai-browser-agent

# 2. Установить зависимости
go mod tidy

# 3. Установить браузер
go build ai-browser-agent/cmd/playwright

# 4. Настроить конфиг (пример в config/local.yml)
Укажите LLM-провайдера, API-ключ, модель и т.д.

# 5. Настроить переменные окружения
Пример настройки вы можете найти в файле .env.example

# 6. Запустить
go build ai-browser-agent/cmd/ai-browser-agent
